### Teknologi yang Digunakan
Proyek ini dikembangkan dengan *stack* teknologi modern untuk memastikan performa dan dokumentasi yang baik:
* **Desain UI/UX:** [Figma](https://www.figma.com/) (Output High-Fidelity)
* **Basis Data:** MySQL (Manajemen Relasional Data)
* **Dokumentasi:** [Docusaurus](https://docusaurus.io/)
* **Version Control:** GitHub
* **Deployment:** Vercel
